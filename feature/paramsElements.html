jQuery Gridy is a plugin that automatically generates a highly customizable grid using templates.

Script:
paramsElements: ['select[name="searchType"]', 'input#extraSearch']
Query string:
search=&page=1&sortName=&sortOrder=asc&find=attribute&rows=10&searchType=advanced&extraSearch=ajax
TIP:
- paramsElements is a way to pass parameters dynamically to your functions;
- Always add the attribute name on your fields, because it is the name of the query string parameter;
- The selectors can be any one accepted by jQuery.
